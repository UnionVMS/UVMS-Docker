
## How to have a running FLUXFMC system in your machine ##

To run the FLUXFMC system you just need to be in the src/main/resources folder and run :

        docker-compose -f docker-compose.yml up -d

NOTE : the images versions are static in this particular docker compose so thay must exist either locally or in docker hub!
